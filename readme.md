# MjGrove

## 概要

Wio LTE/Wio 3Gで利用できる、Groveライブラリです。  
Groveモジュールのクラスと、Grove対応ボードのクラスが用意されており、この2つのクラスを組み合わせるだけでGroveモジュールから値の取得、設定ができます。
