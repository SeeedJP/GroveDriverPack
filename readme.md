# MjGrove

## 概要

[SeeedJP STM32F4 Boards](https://github.com/SeeedJP/SeeedJP_STM32F4)で利用できる、Groveライブラリです。  
Groveモジュールのクラスと、Grove対応ボードのクラスが用意されており、この2つのクラスを組み合わせるだけでGroveモジュールから値の取得、設定ができます。  

まだ開発に着手したばかりなので、互換性のない**破壊的変更**が発生する場合があります。ご了承ください。
